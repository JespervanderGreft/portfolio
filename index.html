Wie ben ik?
Ik ben Jesper van der Greft en ik ben 24 jaar. Ik studeer Communication and Multimedia Design op de Hogeschool van Amsterdam en zit nu in het 4e jaar.

Tijdens mijn opleiding heb ik meer voor de user experience kant gekozen en het designen van apps. Tijdens mijn lange stage heb ik meer kennis gemaakt met het maken van websites en daar ben ik ook meer mee gaan doen.

Instagram Website namaken

Voor het vak Front-end development moesten we een website namaken met HTML, CSS & Javascript, ik heb er toen voor gekozen om Instagram na te maken. We moesten alle code zelf schrijven en moesten ervoor zorgen dat er zo weinig mogelijk classes werden gebruikt. Dit project heeft er echt voor gezorgd dat ik beter ben geworden in HTML & CSS en het ook leuker ben ga vinden.

Project Web MyTrip

We kregen de opdracht om een app te ontwerpen wat voor een reis, tijdens een reis en na een tijdens gebruikt kan worden. Denk aan Booking.com, Polarsteps enzovoort. Mijn app heet MyTrip en combineert verschillende functies uit verschillende apps en die heb ik op mijn eigen manier in MyTrip gemaakt. De app bestaat uit een 3 prototype's, 1 voor de reis, 2 tijdens de reis en 3 na de reis. Hieronder een link naar het prototype wat je voor de reis kan doen met de app.
